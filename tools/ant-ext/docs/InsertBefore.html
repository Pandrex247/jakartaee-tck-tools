insertbefore

Description:

The insertbefore task traverses through a file looking for a specific string. If the string is located the insertString (or the contents of the insertFile file) is added to the output file directly before the current line the the string was found in. The number of occurrences can be specified to limit how pervasive the inserting is.

Parameters:

Attribute

Description

Required

srcFile

The source file which is read and searched.

Yes

destFile

The destination file which contains the source file contents and any strings added as a result of the inserting.

Yes

occurrence

The destination file which contains the source file contents and any strings added as a result of the inserting. The Default value is "all". The other values that can be specified are: "specific", or "after".

No

index

The index of which string to start the inserting.

Yes, if occurrence is set to "specific" or "after".

searchString

The string to search for in the srcFile.

Yes

insertString

The string to insert once a match is found in the srcFile.

Yes, if insertFile is not specified

insertFile

The contents of a file to insert once a match is found in the srcFile.

Yes, if insertString is not specified

Examples:

    <insertbefore srcFile="/tmp/html_basic.tld" destFile="/tmp/new.tld" occurrence="specific" index="1" searchString="</taglib" insertString="<my-tag>com.sun.ts.tests.jsf.common.tags.SillyArgsExtraInfo</my-tag>" />

    <insertbefore srcFile="/tmp/html_basic.tld" destFile="/tmp/new.tld" occurrence="specific" index="1" searchString="</taglib" insertFile="/tmp/testtag.tld" />
