dotarget

Description:

The dotarget task traverses a source tree looking for build files that contain the specified build task. If the task is located, it is executed, otherwise the build file is skipped.  The order in which subdirs are traversed is undefined.  Normally it is only used in bin/build.xml.

Parameters:

Attribute

Description

Required

srcdir

where to start the traversal

Yes

todo

target to invoke in build files

Yes

includes

includes various directories

No

excludes

excludes various directories

No

Examples:

    <target name="build" depends="compile">
          <dotarget srcdir="${user.dir}" todo="package"/>
    </target>
