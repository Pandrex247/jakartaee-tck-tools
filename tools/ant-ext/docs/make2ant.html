Make2ant


A custom task and tool to convert Makefiles to ant build file.
  • For service tests
    • extract data from all Makefiles in a non-leaf dir
    • populate template build file.
    • conditionally sccs edit build.xml
    • conditionallly sccs create build.xml or sccs delget build.xml
  • For non-service tests
    • extract data from all Makefiles in a non-leaf dir
    • extract ejb classes, interfaces, servlet classes from *props file
    • inspect ear's generated by tsmake for the content
    • inspect war's and jar's inside each ear to determine which classes need to be included using support
    • populate template build file.
    • conditionally sccs edit build.xml
    • conditionallly sccs create build.xml or sccs delget build.xml
