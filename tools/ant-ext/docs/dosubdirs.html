dosubdirs

Description:

dosubdirs task traverses source tree to perform the specified build task in all leaf directories.  It enables us to omit build files in non-leaf directories.  The order in which subdirs are traversed is undefined.  Normally it is only used in bin/build.xml.

Parameters:

Attribute

Description

Required

srcdir

where to start the traversal

Yes

todo

target to invoke in build files in each leaf directory

Yes

failonerror

fail build immediately, otherwise build completes and only lists failed directories at the end. Defaults to false.

No

Examples:

    <target name="build" depends="compile">
          <dosubdirs srcdir="${user.dir}" todo="package"/>
     </target>
