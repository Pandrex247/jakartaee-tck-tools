appendafter

Description:

The appendafter task traverses through a file looking for a specific string. If the string is located the append string is added to the output file directly after the current line the the string was found in. The number of occurrences can be specified to limit how pervasive the appending is.

Parameters:

Attribute

Description

Required

srcFile

The source file which is read and searched.

Yes

destFile

The destination file which contains the source file contents and any strings added as a result of the appending.

Yes

occurence

The destination file which contains the source file contents and any strings added as a result of the appending. The Default value is "all". The other values that can be specified are: "specific", or "after".

No

index

The index of which string to start the appending.

Yes, if occurrence is set to "specific" or "after".

searchString

The string to search for in the srcFile.

Yes

appendString

The string to append once a match is found in the srcFile.

Yes

Examples:

    <appendafter srcFile="/tmp/html_basic.tld" destFile="/tmp/new.tld" occurrence="after" index="30" searchString="<tagclass" appendString="<tei-class>com.sun.ts.tests.jsf.common.tags.SillyArgsExtraInfo</tei-class>" />
