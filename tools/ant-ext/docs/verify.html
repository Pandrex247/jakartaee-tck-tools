verify

Invokes Sun RI verify tool, which statically verify all component archives under a start dirctory, including all subdirectories.
 

Parameters:


Attribute
Description
Required
outputdir
where to write output files.  The default is system temp directory, e.g., /var/tmp in Solaris
No
startdir where to start searching for archives.  The default is the corresponding dist directory of the current directory. No
mergeoutput whether to merge all txt output files into one file (all.txt) under ${outputdir}. The default is true No
reportlevel One of "a", "w", "f", which reports all results, warning and failures, and failures, respectively.  The default is "w" No

EXamples:

<target name="verify">
  <verify/>
</target>
<target name="verify">
  <verify mergeoutput="false" reportlevel="a"/>
</target>
