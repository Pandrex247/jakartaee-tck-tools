String

It provides the similar methods to java.lang.String.  You will need to taskdef it before using this task.
 

Parameters:


Attribute
Description
Required
this
the original string to operate on
Yes
method the method name (as in java.lang.String) to invoke
Yes
return the property to hold the return value
Yes
arg1 first parameter of the method
No
arg2 second parameter of the method
No
arg3 thid parameter of the method
No
arg4 fourth parameter of the method
No
arg5 fifth parameter of the method
No

 

EXamples:

<taskdef name="string" classname="com.sun.ant.taskdefs.common.MyString"/>
<string method="replace" this="ejb, jsp, servlet, jaxp, samples" arg1="," arg2=" " return="result"/>

After this task is executed, the property result is assigned the value " ejb jsp servlet jaxp samples", which is the same as executing the java statement:
String result = this.replace(arg1, arg2);
