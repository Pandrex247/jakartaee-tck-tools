getclient


Given a src dir, getclient tries to figure out the test client class (e.g., HelloTestClient.java).  This information is needed to run a single test in javatest batch mode.  It collects all java source files whose name contains the string "client" or "Client", and select the first one.

Parameters:

Attribute

Description

Required

srcdir

where to search for test client

Yes

The same of the test client java file (e.g., HelloTestClient.java) will be assigned to property test.client.

EXamples:

<target	name="get.client" unless="test.client">
  <getclient srcdir="${ts.home}/src/${pkg.dir}"/>
</target>
