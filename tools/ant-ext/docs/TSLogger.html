TSLogger

Description:

TSLogger filters out output from targets that do not do any actual work, and useless message.
The following messages are filtered out:
  • any message that starts with the string "No local string"
TSLogger is hooked up in tsant/tsant.bat using option -logger com.sun.ant.TSLogger.  It can be enhanced to perform other routines, such as sending email notification on build status, etc.
